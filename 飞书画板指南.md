# Hermes Agent Harness 架构图 — 飞书使用指南

> 本文档配合 `agent-harness-architecture.html` 使用

---

## 方法一：快速截图贴文档（5分钟，不可编辑但高清）

1. **打开 HTML 文件**
   ```bash
   open agent-harness-architecture.html
   ```
   或直接用浏览器（Chrome / Edge / Safari）打开该文件。

2. **保存图片**
   - **Chrome/Edge 推荐方式**：在图表区域 `右键 → 另存为图片`，保存的是 **SVG 矢量图**，放大不会模糊。
   - **通用方式**：直接截图（Mac: `Cmd+Shift+4` / Windows: `Win+Shift+S`）。

3. **插入飞书**
   - 打开飞书文档，直接 `Ctrl+V` 粘贴图片。
   - SVG 文件可先拖入文档，飞书会自动转为高清图片。

---

## 方法二：飞书画板绘制可编辑图（推荐，15分钟）

飞书画板中的图形、文字、连线均可编辑。以下是每张图的**快速复刻指南**，包含每个框的文本、颜色值和连接关系。

### 操作技巧
- 在画板中按 `T` 插入文本框，`R` 插入矩形，`O` 插入圆形。
- 按住 `Cmd/Ctrl` 可以多选，批量改颜色。
- 画板支持直接从别处复制文本粘贴到框内。

### 📊 图1：总体架构图（重点，建议优先画）

**布局**：左右分层，共 5 列

| 列 | 图形 | 文本 | 填充色 | 边框色 |
|---|---|---|---|---|
| 接入层 | 圆角矩形×3 | Web 控制台 / REST API / SDK·CLI | `#f2f3f5` | `#dee0e3` |
| Harness层 | 圆角矩形×4 | API Gateway<br>鉴权/限流/路由 等 | `#e8f3ff` | `#165dff` |
| 运行时层 | 圆角矩形×4 | Agent Factory / Runtime: ReAct 等 | `#e8ffea` | `#00b42a` |
| 能力层 | 圆角矩形×4 | Model Router / Tool Registry 等 | `#fff7e6` | `#ff7d00` |
| 基础设施层 | 圆柱形×4 | Metadata DB / Vector DB 等 | `#f7f8fa` | `#c9cdd4` |

**连线关系**（从上到下）：
- 接入层 3个框 → 全部连到 `API Gateway`
- `API Gateway` → `Session Manager` → `Orchestrator`
- `Orchestrator` → `Task Scheduler`（横向）
- `Orchestrator` → `Agent Factory`（向下）
- `Agent Factory` → 分出3条到 ReAct / Plan-Execute / Custom
- 3个 Runtime 各自向右连到 Model Router / Tool Registry / Memory Service
- Memory Service 向下连 Vector DB 和 Cache
- Task Scheduler 向下连 Message Queue

**画板配色速查**：
- Harness 核心层背景：`#e8f3ff`（浅蓝）
- Agent 运行时层背景：`#e8ffea`（浅绿）
- 能力层背景：`#fff7e6`（浅橙）

- Tool → Observation → 回到 Context Builder（画一个向上回的弧线）
- Sub-task → Sub-Agent 调度 → 回到 Context Builder
- 输出 → Memory Store

**重点**：LLM Call 用深蓝色（`#165dff`，白字），决策用橙色（`#ff7d00`，白字），Tool 用绿色（`#00b42a`，白字）。

**连线**：

- 共享状态池 → Reviewer

## 方法三：使用第三方工具转可编辑（进阶）

如果您需要更专业的效果，可以：

1. **draw.io / diagrams.net**
   - 访问 https://app.diagrams.net/
   - 手动绘制后导出为 `SVG` 或 `PNG`
   - 优点：比飞书画板功能更强，对齐更方便
   - 导出后插入飞书文档效果最佳

2. **ProcessOn / 语雀画板**
   - 支持直接粘贴 Mermaid 代码渲染为可编辑图形
   - 渲染完成后导出图片插入飞书

3. **Mermaid Live Editor（在线渲染高清图）**
   - 访问 https://mermaid.live/
   - 复制 HTML 文件中的 Mermaid 代码粘贴进去
   - 点击 `Actions → PNG/SVG` 导出高清图
   - **优点**：完美还原，无需手动绘制
