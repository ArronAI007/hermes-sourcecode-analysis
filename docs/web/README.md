# web/ 目录解释

`web/` 是 Hermes Agent 的 Web 前端，提供浏览器内访问的用户界面。

---

## 目录结构

```
web/
├─── public/                    # 静态资源
├─── src/
│   ├─── components/            # React 组件
│   ├─── hooks/                 # 自定义 Hooks
│   ├─── lib/                   # 工具函数
│   ├─── pages/                 # 页面
│   ├─── styles/                # 样式文件
│   └─── ...
└─── ...
```

---

## 技术栈

- **框架**：React / Next.js
- **样式**：Tailwind CSS / CSS Modules
- **状态管理**：React Query / Zustand
- **API**：REST / WebSocket
